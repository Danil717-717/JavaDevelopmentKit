Lifecycle  - жизненный цикл maven
- clean      - полностью чистит проект(убирает таргеты)
- validate   - проверяет корректность структуры и зависимостей
   например если зависимости зависят друг ог друга петлей
- compile   - скомпилирует классы(появится таргет)
- test      - запустит юнит тесты если они есть
- package   - собирает проект, так же можно повесить доп
   функции например сделать .jar файлы
- verify
- install   - так же как и package  инсталлирует проект и
   сохраняет на диске, можно потом им воспользоваться
- site
- deploy   - можна наш инстальник залить на какой-нибудь сервер
   и на нем запустится наше приложение
